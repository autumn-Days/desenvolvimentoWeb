Criando um site de receitas

Descrição

Neste exercício seu objetivo é criar uma página web que exibe uma receita de um prato de sua escolha.

  1. Crie um novo arquivo HTML e salve-o com um nome adequado, como “receita.html”.
  2. Use a tag <html> para definir o documento HTML
  3. Use a tag <head> para incluir uma tag de título com o título da página de receita.
  4. Use a tag <body> para definir o corpo do seu documento.
  5. Use a tag <h1> para criar um título principal para a página de receita.
  6. Use a tag <p> para criar um parágrafo que apresenta a receita e descreve sua origem ou contexto cultural.
  7. Use a tag <h2> para criar um sub-título para a seção de ingredientes.
  8. Use a tag <ul> para criar uma lista dos ingredientes necessários para a receita. Use as tags <li> para adicionar cada ingrediente à lista.
  9. Use a tag <h2> para criar um sub-título para a seção de instruções.
  10. Use a tag <ol> para criar uma lista de instruções sobre como preparar o prato. Use as tags <li> para adicionar cada etapa à lista.
  11. Use a tag <img> para adicionar uma imagem do prato preparado à página. Você pode usar uma imagem de estoque gratuita ou tirar uma foto do prato você mesmo.
  12. Use a tag <p> para escrever uma breve declaração final, como uma sugestão para o que o prato pode ser servido junto, ou uma anedota pessoal relacionada ao prato.

Certifique-se de validar seu código HTML usando um validador como o W3C Markup Validation Service, para garantir que seu código esteja sem erros e bem formado.
